# Contributing to jakebellacera.com

All contributions are welcome!

## Getting your local environment set up

We use [pnpm](https://pnpm.io) and [Astro](https://astro.build). The setup instructions are typical for Astro sites:

1. Install dependencies `pnpm install`
2. Run dev server: `pnpm dev`

## Getting your changes to production

Any changes merged to `main` will get deployed to production. Just open a PR and if approved and merged, your changes will be live.
